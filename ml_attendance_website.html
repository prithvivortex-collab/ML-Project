About the Project

A Complete ML Pipeline for Attendance Analysis

This project focuses on applying machine learning to analyze and predict student attendance using academic and behavioral attributes such as study hours, sleep patterns, and travel time. The system helps institutions proactively identify at-risk students and improve academic engagement.

01
Data Preprocessing & Cleaning
Removed irrelevant columns, encoded categorical variables, and applied StandardScaler feature scaling.
02
Regression & Classification
Supervised models to predict attendance value and classify High vs Low attendance categories.
03
Clustering Analysis
Unsupervised grouping using K-Means, Hierarchical, and DBSCAN to find hidden student behavior patterns.

Phase 3 — Unsupervised Learning

Clustering Analysis

Clustering grouped students by behavioral similarity without using attendance labels, revealing hidden student segments.

K-Means Clustering

Structured Grouping

n_clusters = 3. Groups students into 3 balanced segments based on feature similarity. Clean, structured cluster distribution.

Hierarchical (Agglomerative)

Nested Merging

Builds clusters step by step by merging similar data points. Produces a dendrogram tree. Uneven but valid distribution.

DBSCAN

Density-Based Detection

eps = 1.5, min_samples = 5. Detects dense clusters and labels irregular outlier students as noise (label -1).

PCA Visualization

Principal Component Analysis reduced the high-dimensional feature space to 2 components — enabling clear visual separation of student clusters.

  • Cluster 0 — High-effort students
  • Cluster 1 — Moderate students
  • Cluster 2 — At-risk students
  • Noise — Outlier points (DBSCAN)
